CNN 출력길이 = ( (입력길이 - 여과기 길이) / 이동크기 ) + 1;

일단 여기서 이동크기는 무조건 1로 하겠다. 고로

CNN 출력길이 = (입력길이 - 여과기 길이) + 1;

이렇게 계산하면 된다.



원본손실을 방지하기 위한 채우기 연산 공식

우선 출력길이와 입력길이를 동일시하게 본다.

C = (N - F) + 1

여기서 N에 대하여 이항을 시킨다.

C + F -1 = N

그리고  C를 원본크기로 하고 N을 채우기 크기로 치환한다.

채우기 (Padding) = 원본 + 여과기 크기 - 1


채우는 시작점은 그 변화량에서 시작한다.

delta = 채우기 - 원본;
